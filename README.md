# FAST-NUCES C++ Fundamentals 💻

This repository contains a collection of core C++ logic, mathematical algorithms, and pattern-solving tasks completed during my 2nd semester at FAST-NUCES Multan.

---

### 📁 Assignment 01: Logic,Loops,Pattrens,Algorithms & Math 
*(University Course Assignment 3)*

This collection focuses on mastering control structures (while/for loops) to solve mathematical and visualization problems.

**Key Problems Solved:**
* **Mathematical Optimization:** Using nested loops to find the minimum points of complex functions like $f(x,y)=(1-x)^{2}+100(y-x^{2})^{2}$.
* **Advanced Algorithms:** Implementation of the **Euclidean Algorithm** for Greatest Common Divisor (GCD) and approximating **Pi ($\pi$)** using infinite series.
* **Number Systems & Logic:** * Binary to Decimal conversion algorithms.
    * Integer manipulation, including reversing numbers and digit summation.
    * Combinatorial logic for the **PakAsia Currency** exchange problem ($6a+9b+20c=n$).
* **Data Visualization:** * Generating dynamic histograms (bar charts) using asterisks.
    * Complex geometric patterns and increasing indentation structures ("Go Buddy Go").

### 🛠️ Tech Stack
* **Language:** C++
* **Environment:** GCC / MinGW Compiler
* **Key Concepts:** Nested Loops, Mathematical Series, Algorithmic Efficiency, Logic Building.
